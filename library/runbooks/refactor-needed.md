# Refactor Needed

Evaluates whether the user's request requires code refactoring without behavior changes.

**True when:** The request involves restructuring, renaming, extracting, or reorganizing code without changing functionality.
**False when:** The request involves adding features, fixing bugs, or changing existing behavior.
