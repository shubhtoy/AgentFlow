# Debug Needed

Evaluates whether the user's request requires debugging or diagnosing a failure.

**True when:** The request involves fixing bugs, diagnosing errors, understanding failures, or resolving test issues.
**False when:** The request is about new features, explanations, exploration, or refactoring.
