The studio is a visual editor for designing, editing, and testing AgentFlow workflows. It runs in the browser and operates directly on your `.agentflow/` directory, so every change you make in the UI is written to disk as plain markdown and YAML files.

## Launching the Studio

<Steps>
<Step>
### Install the CLI

```bash
npm install -g agentflow
```
</Step>
<Step>
### Open a workspace

Navigate to a directory that contains an `.agentflow/` folder (or where you want to create one) and run:

```bash
agentflow studio
```

The studio opens at `http://localhost:4200` by default. Pass `--port` to change it.
</Step>
<Step>
### Create a new workspace (optional)

If no `.agentflow/` directory exists, the studio shows a setup wizard that scaffolds the directory structure for you. You can also run `agentflow init` from the terminal first.
</Step>
</Steps>

## Main Areas

The studio interface is divided into four regions:

### Canvas (center)

The main editing surface where you build workflows as directed graphs. Drag nodes, draw edges, and arrange layouts visually. The canvas supports panning, zooming, multi-select, and right-click context menus.

### Sidebar (left)

The sidebar contains the **Explorer** (file tree of your `.agentflow/` workspace) and the **Elements** panel (resources grouped by category). Toggle the sidebar with `Cmd+B`.

### Panels (right and bottom)

Context-sensitive panels open alongside the canvas. The **Editor** and **Frontmatter** panels appear when you select a node. The **Validation**, **Tokens**, **Git**, **MCP**, and **Copilot** panels are available from the toolbar or keyboard shortcuts.

### Status Bar (bottom)

Displays the active workflow name, current branch, validation summary (error and warning counts), and total token usage. Click any status bar item to open its corresponding panel.

<Callout type="info">
Every panel can be resized, collapsed, or detached into a floating window. Drag a panel header to reposition it, or double-click to toggle between docked and floating modes.
</Callout>

## Keyboard Quick Reference

| Shortcut | Action |
|----------|--------|
| `Cmd+K` | Command palette |
| `Cmd+B` | Toggle sidebar |
| `Cmd+,` | Settings |
| `Cmd+Shift+E` | Export dialog |
| `Cmd+S` | Save current file |
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |
| `Delete` | Remove selected node or edge |
| `Space` (hold) | Pan mode on canvas |
