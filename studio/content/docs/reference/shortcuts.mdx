---
title: Keyboard Shortcuts
description: All keyboard shortcuts in the studio.
---

| Shortcut | Action | Context |
| --- | --- | --- |
| `Cmd+K` | Command palette | Global |
| `Cmd+S` | Save | Editor |
| `Cmd+Shift+V` | Validate | Global |
| `Cmd+Shift+E` | Export dialog | Global |
| `Cmd+,` | Settings | Global |
| `Cmd+B` | Toggle sidebar | Global |
| `Cmd+N` | New file | Global |
| `Cmd+D` | Duplicate node | Canvas |
| `Del/Backspace` | Delete selected | Canvas |
| `Cmd+G` | Git panel | Global |
| `Scroll` | Zoom | Canvas |
| `Drag` | Pan | Canvas |
| `Click` | Select | Canvas |
| `Double-click` | Edit node | Canvas |
| `Right-click` | Context menu | Canvas |
| `Esc` | Close dialog/deselect | Global |
| `Arrow keys` | Navigate | Command palette |
