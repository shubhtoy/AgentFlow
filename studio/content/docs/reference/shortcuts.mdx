<Callout type='info'>
On Windows/Linux, replace `Cmd` with `Ctrl` for all shortcuts below.
</Callout>

<Tabs items={['Global', 'Canvas', 'Editor', 'Panels']}>
<Tab value='Global'>

## Global Shortcuts

These work anywhere in the studio.

| Shortcut | Action |
| --- | --- |
| `Cmd+K` | Command palette |
| `Cmd+Shift+V` | Validate |
| `Cmd+Shift+E` | Export dialog |
| `Cmd+,` | Settings |
| `Cmd+B` | Toggle sidebar |
| `Cmd+N` | New file |
| `Cmd+G` | Git panel |
| `Esc` | Close dialog/deselect |

</Tab>
<Tab value='Canvas'>

## Canvas Shortcuts

These work when the workflow canvas is focused.

| Shortcut | Action |
| --- | --- |
| `Cmd+D` | Duplicate node |
| `Del/Backspace` | Delete selected |
| `Scroll` | Zoom |
| `Drag` | Pan |
| `Click` | Select |
| `Double-click` | Edit node |
| `Right-click` | Context menu |

</Tab>
<Tab value='Editor'>

## Editor Shortcuts

These work when the markdown editor is focused.

| Shortcut | Action |
| --- | --- |
| `Cmd+S` | Save |

</Tab>
<Tab value='Panels'>

## Panel Shortcuts

Shortcuts for navigating panels and palettes.

| Shortcut | Action |
| --- | --- |
| `Arrow keys` | Navigate (command palette) |

</Tab>
