## Planned Features

**On-platform AI execution** — Run workflows directly on supported platforms instead of only exporting static definitions. This enables live debugging and iterative development without leaving the studio.

**Real-time collaboration** — Multiple users editing the same workflow simultaneously with presence indicators and conflict resolution.

**More platform targets** — Expanding export support beyond LangGraph, Claude, and GitHub Actions to additional orchestration frameworks and agent runtimes.

**Plugin system** — A standard interface for extending AgentFlow with custom nodes, exporters, validators, and integrations.
