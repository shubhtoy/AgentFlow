## Canvas

The workflow graph. Nodes are steps (▪) or routers (◇). Green border marks entry points. Edges show execution flow — conditional edges are labeled with their runbook.

<ComponentPreview height="lg">
  <DocsPlayground panel="none" />
</ComponentPreview>

Drag to reposition • Scroll to zoom • Click to select • Double-click to edit • Right-click for context menu

## Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+K` | Command palette |
| `Cmd+S` | Save |
| `Cmd+Shift+V` | Validate |
| `Cmd+Shift+E` | Export |
| `Cmd+,` | Settings |
| `Cmd+B` | Toggle sidebar |
| `Cmd+N` | New file |
| `Cmd+D` | Duplicate node |
| `Del` | Delete selected |
| `Cmd+G` | Git panel |
