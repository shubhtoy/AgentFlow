# Checkpoint

Pause and confirm with the user before proceeding to the next phase.

## What to present
- Summary of what was just completed
- Current status: what's done, what's next
- Any issues encountered and how they were resolved
- Ask: "Ready to continue?"

## User options
- **Confirm** — continue to the next task or phase
- **Cancel** — stop and discuss before proceeding
