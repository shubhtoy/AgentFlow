# Confirm Destructive Action

Pause before executing a destructive or irreversible operation. Present the action and its consequences clearly.

## What to present
- Exact description of the destructive action
- What will be affected (files, data, services)
- Whether the action is reversible
- Suggested alternatives if available

## User options
- **Confirm** — proceed with the destructive action
- **Cancel** — abort and discuss alternatives
