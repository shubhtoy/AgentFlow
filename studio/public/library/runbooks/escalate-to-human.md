# Escalate to Human

The agent has encountered a situation that requires human judgment. This is not a failure — it's a responsible handoff.

## When to escalate
- Security-sensitive decisions (access control, encryption choices)
- Ambiguous requirements that could go either way
- Breaking changes that affect other teams
- Performance trade-offs with no clear winner
- Anything involving production data or deployments

## What to present
- Clear description of the decision needed
- Options considered with pros/cons for each
- The agent's recommendation (if it has one)
- What information would help resolve the ambiguity
